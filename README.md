## Quick Start

### Prerequisites
- Java 21+
- PostgreSQL (for production)

### Development

Clone and build:

```bash
git clone https://github.com/AnnaChekina/java-project-72
cd java-project-72/app
make build
```
Run the application:

```bash
make run
```
App will be available at: http://localhost:7070

### Testing

Run tests:

```bash
make test
```
Run tests with coverage:

```bash
make report
```
Run checkstyle

```bash
make checkstyle
```
